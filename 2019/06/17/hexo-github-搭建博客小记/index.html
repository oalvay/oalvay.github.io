hexo+github 搭建博客小记

搭建这个博客是临时起意，用了差不多半天，比预计的时间要短。想当初学python，学C，学用kaggle，学用VNR打黄油…其中相当一部分给我带来了长期收益，故以后学新工具都将归类到“挤多不压身”的标签之下。

以前有试着用django搭博客，当时奋战到半夜结果只搞了个开头，心情实属郁闷。不过那次经历算是让我初步了解了HTML语法，也之后学爬虫打下了一点点基础。相比之下，这次选用hexo+github简单了很多，不需要自己写代码，大部分都直接在终端里命令就可以了。

参考的材料

https://hexo.io/docs/
参考了安装和初始化部分。
https://www.jianshu.com/p/189fd945f38f
搭建博客主要参考地。

另外还有一些参考了一小部分的网站，其中大部分都是csdn和简书的。

此外主题的设置可在 /theme 里主题的 _config.yml 里修改。

遇到的难点

  • Error: EACCES: permission denied

    开始我以为我是以非管理员身份在输入命令，但提升文件修改权限之后还是一样。最后在网上查靠

解决了问题。而普通的权限问题在命令前加 sudo 就能解决。

  • 本地预览和发布效果不一样

    这个貌似在修改 _config.yml 里的url一栏后就解决了。

  • categories和tags页面无内容

    我是在用Nlvi主题遇到的这个问题。categories无内容的原因可能是Nlvi本身不支持categories，因为在landscape主题里显示正常。tags和其他一些问题是由于Nlvi主题文件里 _config.yml 被命名成 _config.template.yml，修改名称后解决。

小结与留坑

这次学习新技能的难度为中等，不过从早上连续学到下午感觉蛮累。

生成了ssh key但暂时啦懒得学咋用，还有多终端同步写博客的坑以后填，参考：
